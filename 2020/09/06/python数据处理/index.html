本地数据处理

打开本地文件with open('/Users/yindelin/Desktop/test.txt') as f:，这样写就不用自己写close来关闭文件

其中open方法的第二个字符串参数代表mode，例如r读文件（文件必须存在）、w清空来写文件/新建一个文件、a追加；默认是r

在读写的时候有一个文件指针，例如调用writelines来写入文件的时候，文件指针已经到达文件末尾，这时候再读取的话，什么都读不出来，为了移动文件指针，可以调用file_obj.seek(offset,start=0)，表示将文件指针移到start+offset字节的位置

../表示上层目录

./表示当前目录
